这篇主要演示，如何在win10 pro下，使用docker安装常用的中间件

环境：

  1. Windows 10 专业版
  2. 安装了docker desktop for windows 4.0
  3. 所有中间件，外部主机的挂载盘路径：/E/dev/dockerdata

安装redis

1. 拉取redis镜像

上面没有指定redis tag，那么拉取到的redis版本是多少，用下面命令查看（在linux环境下可以，win10 powershell无法没有grep命令）：
docker image inspect redis:latest | grep -i version

2. 运行镜像：

查看容器启动日志：
docker logs -f myredis

3. 在docker下运行redis客户端，就能使用命令来测试

docker exec -it myredis redis-cli

4. 客户端连接测试：

电脑关闭重启或重启docker后，如果没有设置自动启动容器。通过docker ps -a可以看到所有关闭的容器，启动容器。
docker ps -a #看到关闭的容器
docker start myredis

安装jenkins(官方jenkins)

1.拉取镜像

docker pull jenkins/jenkins:lts

2.运行镜像

docker run -d –name myjenkins -p 8081:8080 -v /e/dev/dockerdata/jenkins:/var/jenkins_home jenkins/jenkins:lts

3.访问jenkins控制台

http://localhost:8081/login

3.1 解锁 Jenkins

页面提示需要到：/var/jenkins_home/secrets/initialAdminPassword 下找到初始密码
可以到容器里面查找这个文件：
docker exec -it jenkins/jenkins:lts bash
more /var/jenkins_home/secrets/initialAdminPassword

也可以到外部挂在目录找到这个文件：
E:\dev\dockerdata\jenkins\secrets 下的 initialAdminPassword

输入完毕初始密码后，进入安装插件页面，选择：安装推荐插件
开始漫长的推荐插件安装

创建管理员用户：
xiexugang / xiexugang

http://localhost:8081/
用户名：xiexugang
密码：xiexugang
