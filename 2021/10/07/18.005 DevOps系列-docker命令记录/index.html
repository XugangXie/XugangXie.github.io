1.docker volume

参考：
https://www.jianshu.com/p/8c22cdfc0ffd
http://dockone.io/article/128

1.1 创建volume：docker volume create --name [volumes名字]

如：docker volume create --name haha
--name可不写

1.2 查看已有的volume：docker volume ls

看到上图中有volume有一些是自己命名的有一些是一串数字：

  • 自己命名的是使用docker volume create --name [volume name]创建的，比如haha 是使用docker volume create --name haha
  • 数字Volume是在创建container的同时创建的

1.3 查看volume在docker虚拟机中的位置：docker volume inspect [volumes名字]

如：docker volume inspect haha

1.4 删除volume：docker volume rm [volumes名字]

如：docker volume rm haha

2.docker network

2.1 docker network ls 查看网络列表

2.1 docker network create创建网络

为zookeeper创建docker网络: docker network create zookeeper_network

2.3 docker network相关命令

命令 说明
docker network ls 列出网络
docker network create xx 创建一个网络xx
docker network rm xx 删除一个网络xx
docker network inspect xx 显示网络xx的详细信息

2.4 docker run –network=xx 使用

启动容器时，使用–network标志将其连接到网络。 此示例将busybox容器添加到mynet网络
docker run -itd --network=mynet busybox

2.5 docker run –hostname=xx 使用

https://www.cnblogs.com/yangmingxianshen/p/10153508.html

docker run --hostname=xx 或 docker run -h xx
-h HOSTNAME或–hostname=HOSTNAME。设定容器的主机名，它会被写到容器内的/etc/hosts和/etc/hostname文件中。
但是这个主机名只有在容器内才能看到，在容器外看不到，既不会在docker ps中显示，也不会在其它容器的/etc/hosts中看到。

3. docker cp

作用：从运行的容器中，拷贝目录和文件到宿主机的命令
语法：docker cp {容器名}:{容器内目录或文件} {宿主机目录或文件}
例子：
docker cp myemqx:/opt/emqx/etc E:\dev\dockerdata\emqx\etc
docker cp myemqx:/opt/emqx/lib E:\dev\dockerdata\emqx\lib
docker cp myemqx:/opt/emqx/data E:\dev\dockerdata\emqx\data
docker cp myemqx:/opt/emqx/log E:\dev\dockerdata\emqx\log

1. docker-compose命令

1.1 启动服务

docker-compose -f docker-compose.yml up -d

1.2 停止服务

docker-compose -f docker-compose.yml stop

1.3 停止并删除服务

docker-compose -f docker-compose.yml down
