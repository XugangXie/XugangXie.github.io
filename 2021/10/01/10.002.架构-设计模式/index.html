Java设计模式分为 创建模式, 结构模式, 行为模式 3种类型：

  • Java设计模式之创建模式包括： 5个
    Factory, 工厂模式

    工厂模式里面的，有些人分为：简单工厂模式、抽象工厂模式
    有些人分为：工厂方法模式、抽象工厂模式

    Singleton, 单例模式
    Builder, 建造模式
    Prototype, 原型模式

  • Java设计模式之结构模式包括： 7个
    Facade, 外观/门面模式 ✔
    Decorator, 装饰器模式 ✔
    Composite, 合成/组合模式
    Proxy, 代理模式 ✔
    Adapter, 适配器模式 ✔
    Bridge, 桥梁/桥接模式 ✔
    Flyweight, 共享元类模式

  • Java设计模式之行为模式包括： 11个
    Chain of responsiblity, 责任链模式 ✔
    Observer, 观察模式 ✔
    State, 状态模式 ✔
    Template, 模板模式 ✔
    Strategy, 策略模式 ✔
    Command, 命令模式
    Interpreter , 解释器模式
    Iterator, 迭代模式
    Mediator, 中介模式
    Memento, 备忘录模式
    Visitor, 访问者模式

23种设计模式 可以归纳为一下三类：

Creational Pattern 创建型模式
1. Singleton Pattern 单例模式
2. Factory Pattern 工厂模式（工厂方法模式）
3. Abstract Factory Pattern 抽象工厂模式
4. Builder Pattern 构造器模式
5. Prototype Pattern 原型模式

Structural Pattern 结构型模式
1. Adapter Pattern 适配器模式
2. Facade Pattern 门面模式/外观模式
3. Decorator Pattern 装饰器模式
4. Composite Pattern 组合模式
5. Proxy Pattern 代理模式
6. Bridge Pattern 桥梁模式
7. 共享元模式

Behavioral Pattern 行为型模式
1. Observer Pattern 观察者模式
2. Chain of Responsibility Pattern 责任链模式
3. Strategy Pattern 策略模式
4. 模板模式
5. Iterator Pattern 迭代器模式
5. Mediator Pattern 中介模式
7. 命令模式
8. 解释器模式
9. 备忘录模式
10. 访问者模式
11. State Pattern 状态模式
