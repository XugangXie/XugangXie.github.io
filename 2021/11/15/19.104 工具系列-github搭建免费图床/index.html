主要演示如何使用github搭建免费的图床
参考：https://zhuanlan.zhihu.com/p/353775844

几个必要条件：

  1. GitHub账号和一个放置图片的仓库：blog-img

  2. 图片上传客户端PicGo，
    下载官网：https://molunerfinn.com/PicGo/
    官网：https://picgo.github.io/PicGo-Doc/zh/guide/#%E5%BA%94%E7%94%A8%E6%A6%82%E8%BF%B0

  3. 免费的CDN加速jsDelivr
    jsDelivr: https://www.jsdelivr.com/?docs=gh

搭建过程：

1. 使用github账号，创建一个空的仓库: blog-img

创建好空的仓库后，接着就是上传图片了

2. 安装 PicGo

下载官网：https://molunerfinn.com/PicGo/
github上下载：https://github.com/Molunerfinn/picgo/releases

3. 配置 PicGo

3.1 首先，我们先要去 Github 创建一个 token

以此打开 Settings -> Developer settings -> Personal access tokens，最后点击 generate new token；

3.2 配置 PicGo，依次打开 图床设置 -> Github 图床；

设定仓库名：XugangXie/blog-image
设定Token：就是上面那步获得的
设定自定义域名：https://cdn.jsdelivr.net/gh/XugangXie/blog-img
因为Github 的图片有时候访问太慢了，有时候甚至直接加载不出来，所以这里使用的是免费的CDN加速 jsDelivr

3.2 测试：

本地拖拽几个图片到 PicGo：

点击图片下面的小笔图标，获取公网链接： https://cdn.jsdelivr.net/gh/XugangXie/blog-img/galaxy1.jpg
