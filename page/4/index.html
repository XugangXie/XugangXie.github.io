2.NC166连续子数组的最大和二(DP、双指针)

该题分析与实现：https://blog.nowcoder.net/n/55045673a518403bb33f72a1034105b8

用dp数组表示以下标i为终点的最大连续子数组和，则每次遇到一个新的数组元素，连续的子数组要么加上变得更大，要么它本身就更大，
因此状态转移为dp[i]=max(dp[i−1]+array[i],array[i])，这是最基本的求连续子数组的最大和。

这题要比NC19连续子数组的最大和难点，难点在于：

  1. 确立了转移方程后，要用两个双指针left、right，用于表示最大和的区间范围；要找到区间起点的判断判断逻辑：array[i] > dp[i - 1] + array[i]，那么left设置新起点
  2. 更新最大值，判断逻辑有②中任意一个，①当dp[i]>maxsum时；②当dp[i] == maxsum && (right - left + 1) > (resr - resl + 1)时，要更新maxsum的值；更新resl = left; resr = right;

算法思想

分治：分治即“分而治之”，“分”指的是将一个大而复杂的问题划分成多个性质相同但是规模更小的子问题，子问题继续按照这样划分，直到问题可以被轻易解决；“治”指的是将子问题单独进行处理。经过分治后的子问题，需要将解进行合并才能得到原问题的解，因此整个分治过程经常用递归来实现。
二分法也属于分治思想的一种实现。
归并也属于分治思想的实现。
